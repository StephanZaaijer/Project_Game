◆ update()

void MainGameState::update ( )
overridevirtual

This function updates all game events in this state.

This function calls the character update function. It calls the collideWalls function on the character to check if the character is colliding with a wall. When the player is above a certain height, score is added to the player's score, the background is moved down, and a new background may be rendered. It also checks if the character has died, it spawns new obstacles if needed and checks if a coin is collected.

Reimplemented from GameState.
